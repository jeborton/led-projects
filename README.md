# LED Projects

This repository contains LED programming and build documentation for:
- Float lighting projects
- Wearable lighting projects

## Main sections
- `float-lights/` — float-specific lighting systems and code
- `wearables/` — wearable and costume lighting projects
- `docs/` — shared repo conventions and workflow notes

## Working rules
- Stable code goes in `stable/`
- New experiments go in `experimental/`
- Retired but useful versions go in `archived/`
- Each project should have a README and supporting notes

## Current active projects
- Float: KOSS
- Wearables: Top Hat, Captain's Hat
