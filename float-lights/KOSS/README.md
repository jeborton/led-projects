# KOSS

KOSS is the current active float lighting project.

## Platforms in use
- Pixelblaze
- CircuitPython
- Arduino

## Main folders
- `docs/` — layout, wiring, and hardware notes
- `pixelblaze/` — Pixelblaze pattern code
- `circuitpython/` — Python-based controller code
- `arduino/` — Arduino sketches and related files
- `config/` — layout and hardware profile references
- `assets/` — diagrams, maps, and photos
